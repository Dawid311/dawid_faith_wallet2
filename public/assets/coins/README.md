# Coin Images Placeholder

Dieser Ordner enthält die Coin-Bilder für die Anwendung.

## Benötigte Bilder:

### Custom Tokens:
- **dfaith.png** - D.FAITH Token Logo (empfohlene Größe: 256x256px)
- **dinvest.png** - D.INVEST Token Logo (empfohlene Größe: 256x256px)

### Format-Empfehlungen:
- **Format:** PNG mit transparentem Hintergrund
- **Größe:** 256x256 Pixel (optimal für alle Display-Größen)
- **Qualität:** Hochauflösend für Retina-Displays
- **Design:** Kreisförmig oder quadratisch (wird automatisch rund dargestellt)

### Platzierung:
```
/public/assets/coins/
├── dfaith.png    <- D.FAITH Token Logo
└── dinvest.png   <- D.INVEST Token Logo
```

**Hinweis:** Bekannte Token (POL, MATIC, USDC, etc.) werden automatisch von CDNs geladen.
